**1. 实时控制面板**

打开浏览器你会看到左预览 + 右控制面板。拖控件、点色块、调字号——左边的公众号文章**实时切换**视觉。

### 四组配置覆盖所有可调项

- **品牌**：品牌名 / 章节小戳 / 期号
- **配色**：主色 / 高亮 / 辅助 / 正文（4 个大色卡）
- **排版**：正文字号 / 行高 / 章节大数字字号
- **图片**：圆角 / 边框宽度 / 边框颜色 / 阴影强度

调到满意，点右下「复制公众号 HTML」一键到剪贴板。

**2. 三种吸色方式**

不同浏览器 + 不同场景，覆盖所有可能：

| 方式 | 兼容性 | 场景 |
|------|--------|------|
| 屏幕吸色 | Chrome / Edge | 鼠标变十字吸全屏任意位置（含其他应用窗口） |
| 图片吸色 | 全浏览器 | 上传 logo / 截图 → 点像素拾取 |
| 粘贴 hex | 全浏览器 + 全工具 | 配合 macOS 数码测色计 / Sip / Figma，吸完 Cmd+V 自动识别 |

### Safari 用户推荐姿势

开 macOS 自带「数码测色计」（Spotlight 搜「Digital Color Meter」），鼠标悬停取色后回到 preview 页**任意位置 Cmd+V**，自动识别 hex 弹应用色板。比浏览器原生 EyeDropper 更精确。

**3. 9 条公众号 HTML 输出规范**

这是 200+ 篇公众号反复粘贴验证沉淀的硬规则，违反 = 公众号渲染崩：

1. 列表用 `<section>+<span>`，禁用 `<ul>/<li>`（公众号会拆行）
2. 加粗用 `<span style="font-weight:700">`，禁用 `<strong>`（后跟全角字符会强插换行）
3. 图片 `max-width:100% + height:auto`，不用 `width:100%`（小图会被拉伸模糊）
4. H3 用左侧主色竖条 + 17px 粗体（视觉节奏统一）
5. 超长 PNG 自动 scrollbox（文件名含「全图 / 长图」触发）
6. 多行引用块用 `<br>` join，不用空格 join（保留换行节奏）
7. frontmatter 中文 key：`标题:` / `副标题:` / `创建时间:` / `标签:`
8. 复制时块级元素剥 width（让公众号容器决定宽度）
9. 复制时根标签用 `<section>` 不用 `<article>`（公众号对 article 有 CSS reset）
